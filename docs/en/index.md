## What is RELS?

RELS (Real Estate Legal Solutions) is a web application for managing real estate deals and legal cases. It replaces the legacy spreadsheet system with a modern interface for lawyers, assistants, and clients.

## Key Features

- **Deal and case management** with a clear dashboard
- **Deal phase tracking** and legal process monitoring
- **Contact database** for clients, counterparties, and banks
- **Escrow and financial management** with full audit trails
- **Integration with government registries** (ARES, RUIAN, Cadastral)
- **Attendance and task management** for the team

## User Roles

RELS supports three roles with different levels of access:

| Role | Description |
|------|-------------|
| **Lawyer** | Full access to all modules, user management, and settings |
| **Assistant** | Access to deals, cases, tasks, contacts, and attendance |
| **Client** | Read-only access to their own deals and documents |
