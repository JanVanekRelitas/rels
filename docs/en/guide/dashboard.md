# Dashboard

The Dashboard is the first screen you see after logging in. It provides an overview of your active work, upcoming deadlines, and quick access to the most important modules.

## Navigating to the Dashboard

- The Dashboard loads automatically after login.
- You can return to it at any time by clicking the **RELS logo** or **Dashboard** link in the sidebar navigation.

## Summary Cards

The top section of the Dashboard displays summary cards with real-time counts:

| Card | Description |
|------|-------------|
| **Active Deals** | Number of deals currently in progress (not completed or archived) |
| **Active Cases** | Number of open legal cases |
| **Pending Tasks** | Tasks assigned to you that are not yet completed |
| **Upcoming Deadlines** | Tasks and deal milestones due within the next 7 days |

::: tip
Click any summary card to jump directly to the filtered list for that category.
:::

## Recent Activity

Below the summary cards, the Dashboard shows a timeline of recent activity across the office:

- New deals and cases created
- Tasks completed or reassigned
- Escrow deposits received
- Status changes on deals and cases

Each activity entry shows the **user**, **action**, **target item**, and **timestamp**.

## Quick Actions

The Dashboard provides quick-action buttons for common tasks:

- **New Deal** — Opens the deal creation form with SP.ZN. 500xxx prefix
- **New Task** — Opens the task creation dialog
- **New Contact** — Opens the contact creation form

## Filtering and Search

- Use the **search bar** at the top of the page to search across deals, cases, contacts, and tasks.
- Search works by SP.ZN. (file reference number), name, or keyword.
- Results are grouped by category (Deals, Cases, Contacts, Tasks).

## Sidebar Navigation

The left sidebar provides navigation to all modules:

- **Dashboard** — Home screen (current page)
- **Deals** — Real estate transactions
- **Cases** — Legal proceedings
- **Tasks** — Task management
- **Contacts** — People and organizations
- **Escrow** — Escrow accounts
- **Finance** — Financial overview
- **Attendance** — Time tracking
- **Registries** — ARES, RUIAN, and Cadastral lookups
- **Settings** — Administration (lawyer role only)

::: warning
The modules visible in the sidebar depend on your user role. Clients see only the Dashboard and their own Deals.
:::

## Personalization

- The Dashboard remembers your last-used filters within the current session.
- Summary cards update in real time as data changes in Firestore.
- The activity feed refreshes automatically without needing to reload the page.
